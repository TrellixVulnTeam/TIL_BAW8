### 부분집합 

- **1 << n  => 원소가 n개인 경우 모든 부분집합의 수  2^n**

- **i & (1 << j )  =>  i의  j 번째 비트가 1인지 아닌지 여부를 리턴**

  1 << 6 = 1000000 = 000001 ~ 111111(2)

```python
    count = -1

    for i in range(1 << N):
        total = 0
        for j in range(N):
            if i & (1 << j):
                total += arr[j]
        print()

        if total == 0:
            count += 1
```
