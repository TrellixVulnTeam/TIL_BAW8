### 명령어

- vue cli 설치

```
npm install -g @vue/cli
```

- 프로젝트 생성 및 이동

```
vue create my-first-app
cd my-first-app
```

- 패키지 설치( github 에서 내려받은 경우 package-lock 설치)

```
npm i 
```

- 뷰 라우터 설치

```
vue add router
```

- axios 설치

```
npm install --save axios
```

- lodash 설치

```
npm install lodash
```

- 실행

```
npm run serve
```
