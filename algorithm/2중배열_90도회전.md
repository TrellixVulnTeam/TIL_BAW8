# 90도 회전시키기

- 2중 배열을 오른쪽으로 90도 회전시키기

- 직접 해보면, 열은 행으로, 행은 열로 (이때 인덱스 활용)

```python
input_arr = [list(map(int, input().split())) for _ in range(N)]

rotated_arr = [[0] * N for _ in range(N)]

for i in range(N):
    for j in range(N):
        rotated_arr[i][j] = input_arr[j][N-1-i]
```
